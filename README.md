# LeafKnow

## 项目简介
LeafKnow是一个知识管理与学习平台项目。

## 项目结构
```
LeafKnow/
├── docs/           # 项目文档
├── src/            # 源代码
├── tests/          # 测试文件
├── .gitignore      # Git忽略文件
├── README.md       # 项目说明
└── CLAUDE.md       # Claude Code 指导文档
```

## 开发团队
- 项目负责人：你
- 维护者：你的朋友们

## 协作流程
1. 每个维护者在自己的分支上开发
2. 通过Pull Request向主分支提交代码
3. 由项目负责人进行代码审查和合并
